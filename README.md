# Lokkit

Этот проект был создан с использованием [Angular CLI](https://github.com/angular/angular-cli) 19.0.3. версиии.

## Сервер разработки

Чтобы запустить локальный сервер разработки, запустите команду:

```bash
ng serve
```

Как только сервер запустится, откройте свой браузер и перейдите по ссылке "http://localhost:4200/`. Приложение будет автоматически перезагружаться всякий раз, когда вы измените какой-либо из исходных файлов.
